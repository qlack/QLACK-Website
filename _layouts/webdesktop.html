WEBDESKTOP

QLACK WebDesktop provides a virtual desktop environment in your browser. If your project needs several application windows to be open at once, QLACK WebDesktop allows you to seamlessly integrate all of your application under the same environment. Application open in individual windows, supporting familiar features such as drag, resize, minimise, maximise, etc.
