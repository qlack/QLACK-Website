QLACK Angular

QLACK Angular provides a set of TypeScript libraries for your Angular based application to facilitate everyday tasks in your SPA development. Tasks such as automatic mapping of backend data validation error messages to your reactive forms, or inter-application communication via a PubSub framework are provided via QLACK Angular.
